## Description

Cat breeds api backend based on Nestjs, mysql. And it is integrated with Swagger api document builder, Api document path is "{Host}/api/".
For example, http://localhost:3000/api  at localhost.
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
